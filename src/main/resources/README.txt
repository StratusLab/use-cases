====================
StratusLab Use Cases
====================

This project contains use cases to validate a StratusLab cloud
infrastructure from the client-side.

More information about the StratusLab cloud distribution can be found
at the collaboration's website: http://stratuslab.eu/.

Feedback
========

To provide feedback or to report bugs, please send an email to
support@stratuslab.eu or open an issue on
[GitHub](https://github.com/StratusLab/use-cases/issues).
